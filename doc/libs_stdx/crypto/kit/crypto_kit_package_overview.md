# stdx.crypto.kit

## 功能介绍

crypto.kit 包提供了一套 [CryptoKit](../common/crypto_common_package_api/crypto_common_package_interfaces.md#interface-cryptokit) 的默认实现，提供随机数生成器及解码 DER、PEM 的能力。
