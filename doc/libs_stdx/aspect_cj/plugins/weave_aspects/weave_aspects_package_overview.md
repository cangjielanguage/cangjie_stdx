# stdx.aspect_cj.plugins.weave_aspects

## 功能介绍

weave_aspects 包提供基于注解的 CHIR 织入插件（`WeaveAspects`），自动注册到 PluginManager，在 `executeCHIRPlugins` 执行时对 CHIR 包进行注解驱动的函数织入。
