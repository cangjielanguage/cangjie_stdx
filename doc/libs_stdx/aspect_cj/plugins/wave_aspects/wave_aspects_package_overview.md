# stdx.aspect_cj.plugins.wave_aspects

## 功能介绍

wave_aspects 包提供基于注解的 CHIR 织入插件（`WaveAspects`），自动注册到 PluginManager，在 `executeCHIRPlugins` 执行时对 CHIR 包进行注解驱动的函数织入。
