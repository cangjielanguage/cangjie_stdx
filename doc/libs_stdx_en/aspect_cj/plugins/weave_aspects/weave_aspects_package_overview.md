# stdx.aspect_cj.plugins.weave_aspects

## Feature Introduction

The weave_aspects package provides an annotation-based CHIR weaving plugin (`WeaveAspects`), auto-registered with PluginManager, that performs annotation-driven function weaving on CHIR packages during `executeCHIRPlugins` execution.
