# stdx.aspect_cj.plugins.wave_aspects

## Feature Introduction

The wave_aspects package provides an annotation-based CHIR weaving plugin (`WaveAspects`), auto-registered with PluginManager, that performs annotation-driven function weaving on CHIR packages during `executeCHIRPlugins` execution.
